#
# Jython Readme 
#

This directory contains Jython script examples how jython 
can be used to acces the Virtual Resource System (and Virtual File System). 
To start them, just call jython.sh and provide the *.py script to run. 

For example: 
	$VLET_INSTALL/bin/jython.sh vfsexample.py 

For an API description how to se the VRS/VFS classes, see:
	 VLET_INSTALL/doc/api/index.html
