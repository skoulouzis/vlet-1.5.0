Help for developers

Read the Virtual Resource System API: API
This also includes the Virtual File System (VFS) API. If you want to use the VFS, start with the VFSClient class:VFSClient

Send bug reports

When submitting a bug report, please include the VLET version number.
If possible include an exception stack trace. If no exception was reported and the VBrowser just 'hangs', press CONTROL-\ (Control and Backslash) in the window that started the VBrowser to produce a stacktrace.
